# FED22 Testning - Inlämningsuppgift 2

API-testing with Vitest, axios och jest.

## Instructions

1. `npm install`
2. Start the API for a previous project (Bortakväll)
3. Fill the URL to the API in the `.env`
4. Running tests
    - `npm test`
    - `npm test:verbose`
    - `npm test:ui`
    - `npm coverage`
